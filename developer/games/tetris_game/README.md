# Tetris Battle - 雙人俄羅斯方塊對戰

## 遊戲介紹
雙人即時對戰的俄羅斯方塊遊戲，透過網路連線進行對戰。

## 遊戲類型
- **類型**: GUI (圖形介面遊戲)
- **玩家數**: 2 人對戰
- **開發者**: HW2 Team

## 操作方式
- **左右方向鍵**: 移動方塊
- **上方向鍵**: 旋轉方塊
- **下方向鍵**: 軟降（慢速下降）
- **空白鍵**: 硬降（直接落到底）

## 計分方式
- 軟降: +1 分
- 硬降: +2 分/格
- 消除 1 行: +100 分
- 消除 2 行: +300 分
- 消除 3 行: +500 分
- 消除 4 行: +800 分

## 系統需求
- Python 3.7+
- pygame 庫

## 安裝依賴
```bash
pip install pygame
```

## 啟動方式
由 Game Store 系統自動啟動，或手動執行:
```bash
python3 game.py
```
