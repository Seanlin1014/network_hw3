# Number Bomb (數字炸彈) 遊戲

## 遊戲簡介

數字炸彈是一個經典的多人猜數字遊戲，支援 **3-8 人**同時遊玩。

### 遊戲規則

1. 系統隨機選擇一個 1-99 之間的「炸彈數字」（玩家看不到）
2. 玩家輪流猜數字
3. 每次猜測後，系統會縮小範圍：
   - 如果猜的數字小於炸彈：範圍下限提高
   - 如果猜的數字大於炸彈：範圍上限降低
4. **踩到炸彈的玩家輸了！** 其他玩家都是贏家

### 範例流程

```
初始範圍: 1-99（炸彈是 42）

玩家 A 猜 50 → 太大！範圍變成 1-50
玩家 B 猜 25 → 太小！範圍變成 25-50
玩家 C 猜 40 → 太小！範圍變成 40-50
玩家 A 猜 45 → 太大！範圍變成 40-45
玩家 B 猜 42 → 💣 踩到炸彈！玩家 B 輸了！
```

## 遊戲特色

- ✅ 支援 3-8 人多人對戰
- ✅ 即時範圍更新
- ✅ 歷史記錄顯示
- ✅ 自動輪流機制
- ✅ 房主可控制開始

## 文件說明

- `game.py` - 遊戲客戶端
- `server_game.py` - 遊戲伺服器
- `lpfp.py` - 通訊協議

## 啟動命令

### 伺服器
```bash
python3 server_game.py {port}
```

### 客戶端
```bash
python3 game.py {host} {port}
```

## 上架到 Game Store

在 Developer Client 中上架時，設定如下：

**啟動命令**:
```
python3 game.py {host} {port}
```

**伺服器命令**:
```
python3 server_game.py {port}
```

**遊戲設定**:
- 類型: Multiplayer
- 最大玩家數: 8
- 最小玩家數: 3

## 技術細節

- **通訊協議**: LPFP (Length-Prefixed Framed Protocol)
- **數據格式**: JSON
- **連線方式**: TCP Socket
- **並發處理**: Multi-threading
